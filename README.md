# ECommerce-Sales-Dashboard

**Overview**
This Ecommerce Sales Dashboard is a data visualization project designed to track and analyze online business performance across multiple dimensions. Built with a focus on clarity and actionable insights, it consolidates sales, profit, orders, and customer behavior into a single interactive interface.

**Key Features**
- Summary Metrics: Displays total sales amount, profit, quantity sold, and number of orders.
- Geographic Analysis: Highlights top-performing states (e.g., Maharashtra, Madhya Pradesh, Uttar Pradesh) with bar charts.
- Category Insights: Pie charts show product distribution across Clothing, Electronics, and Furniture.
- Temporal Trends: Month-wise profit visualization reveals seasonal fluctuations and loss periods.
- Customer Analysis: Identifies top 10 customers by purchase value.
- Payment Mode Breakdown: Pie chart comparison of COD, UPI, Credit/Debit Cards, and EMI usage.
- Profitability by Subcategory: Pinpoints high-margin items like Printers, Bookcases, and Sarees.

**Technical Highlights**
- Designed with interactive charts and KPIs for quick decision-making.
- Uses color-coded visualizations to enhance readability and engagement.
- Built to be scalable and adaptable for different datasets.
- Can be integrated with Power BI / Tableau / Python (Matplotlib, Seaborn, Plotly) depending on the tech stack.
